Points

The parse quite likely will match more than one frame - and generate more than one point. Frames are sorted by the number of components with more complex ones listed first. Most parses will match simple frames like NP VERB since a match is qualified when all of the required relations for the frame are matched.

In these cases, for simple frames that the verb contains a larger number of words, this is because they are related to the verb and are not matched by another component in the frame.

Unmatched Frames

These frames were found for the verbs in the sentence, however based on the relations currently defined, they didn't match the tree.

Frames marked 'Missing Representation' have yet to get their relations defined, currently around 95% should be covered however.

Clearly not all frames for all verbs in the sentence should match, but there are likely some that ought to that don't currently.
